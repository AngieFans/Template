1 进程开启后，需要调用LruMap.init()方法后才可以使用。通过LruMap.getInstance()方法获取LruMap对象。

2 每次调用put()方法之前最好先调用get()方法获取对象，如果为null再创建对象，然后调用put()方法保存对象。如果不为null就不要创建对象，直接操作对象即可。

3 如果不是持久化对象，通过get()方法获取到对象后，每次将对象的属性值修改，不需要重新调用put()方法。

4 如果是持久化对象，通过get()方法获取到对象后，每次将对象的属性值修改，需要重新调用put()方法。
因为如果将对象的属性值修改后，没有调用put()方法重新保存到内存和序列化文件，进程重启，再通过get()方法获取到的对象的属性值跟最后修改的值不相同。

5 createOrGetElement()方法会先获取对象,如果为null会自动创建对象同时保存到内存,然后返回该对象。
如果不为null直接返回该对象。至于需不需要持久化存储,在调用的时候传对应的参数即可。该方法完全可以代替get和put的方式，推荐使用。

6 为了确保在多线程环境下不出问题，加上用LruMap也不耗时，请放心在主线程使用。
